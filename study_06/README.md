Decorator
- class Decorator
- method Decorator
- property Decorator
- parameter Decorator

모든 Decorator는 function이다


1. 프로젝트 생성
```
mkdir ts-decorator
cd ts-decorator
npm init
```
2. ts 설치
```
npm i
npm i typescript -D
```
3. tsconfig 설정
```
node_modules/.bin/tsc --init
experimentalDecorators 추가
```
4. vscode 컴파일 설정
```
${workspaceRoot}/node_modeules/.bin/tsc
command + shift + <B>
```
